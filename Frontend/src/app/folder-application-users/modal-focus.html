Das erste fokussierbare Element innerhalb des Modalfensters erhält beim Öffnen den Fokus. Dies kann konfiguriert werden, sodass stattdessen ein anderes Element den Fokus erhält, indem man diesem das Attribut ngbAutofocus hinzufügt. 


	
		
			<button 
			type
			
				=
				"button
				"
			 
				ngbAutofocus 
				class
				
					=
					"btn btn-danger
					"
						
					
				
      (click)
	  
		=
		"modal.close('Ok click')">Ok</button>
